NaN

NaN is the result of invalid operations.

console.log('0/0 is ' + 0/0);

NaN is toxic.

console.log('(NaN + 5) * 2 + 14 is ' + ((NaN + 5) * 2 + 14));

NaN is not equal to anything, including NaN.

console.log('Boolean(NaN === 5) is ' + Boolean(NaN === 5)); console.log('Boolean(NaN === NaN) is ' + Boolean(NaN === NaN)); console.log('Boolean(NaN == NaN) is ' + Boolean(NaN == NaN));

The type of NaN is number.

console.log('typeof NaN is ' + typeof NaN);
